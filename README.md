# Cnidarian_pLGIC
Respository contains 
1) supplementary information for reproducing the phylogenetic trees in the manuscript
2) the R code necessary to reproduce the expression plots for single cell data found in Ojha, Kloss, Montenegra et al. 2026. Expression matrix is available at sea-anemone-atlas.cells.ucsc.edu (https://cells.ucsc.edu/?ds=sea-anemone-atlas+Nv2+all)
| File      | Description | Notes    |
| :---        |    :----:   |          ---: |
| annot_table      | Table containing meta informationof the sequences used in this study       | The annotation table is used in the script in order to improve names   |
| pLGIC.einsi.aln.fasta   | Full protein alignment using MAFFT einsi algorithm        | For reference only      |
| pLGIC.einsi.aln.trim.fasta      | Trimmed protein alignment produced by TrimAl with automatic trimming       | This was used for phylogenetic tree construction   |
| pLGIC.por.einsi.aln.fasta   | Full protein alignment including porifera pLGICs using MAFFT einsi algorithm        | Reference only      |
| pLGIC.por.einsi.aln.fasta      | Trimmed protein alignment with porifera pLGICs produced by TrimAl with automatic trimming       | This was used for phylogenetic tree construction   |
| plot.R   | Script to reproduce phylogenetic tree in the main manuscript        |      |
| scRNA_data_pLGICs.R   | Script to reproduce single Cell analysis and figures in the main manuscript        |      |
